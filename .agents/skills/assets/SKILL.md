# Stellar Assets Guidelines

1. **Classic Assets**: Handle asset issuance, establishing trustlines (`changeTrust` operation), and standard payments.
2. **Soroban Token Interface**: Align with standard token interfaces (e.g. SEP-41 token interface) when writing contract-based custom assets.
