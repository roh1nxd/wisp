# Stellar Standards Guidelines

1. **SEPs (Stellar Ecosystem Proposals)**: Follow standard SEP flows (e.g. SEP-10 for authentication, SEP-24/SEP-6 for deposits/withdrawals, SEP-12 for KYC).
2. **Standard Implementation**: Prefer existing SEP flows over custom implementations to maintain ecosystem interoperability.
